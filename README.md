# ETHOS · PJF Bot Expedientes

Aplicación de escritorio (Electron) para consultar expedientes en el portal **Juicio en Línea** del Poder Judicial Federal y generar un Excel actualizado con el estado procesal.

## Documentación

Toda la guía de uso, desarrollo, variables de entorno, formato del Excel y generación del instalador está en la carpeta **[`doc/`](doc/README.md)**.
